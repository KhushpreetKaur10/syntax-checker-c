# Syntax Checker and C Code Analyzer

A simple C program that analyzes a C source file and reports details such as:

✔ Total lines of code  
✔ Keywords used with line numbers  
✔ Built‑in functions count  
✔ Number of user functions and prototypes  
✔ Variable counts and memory used  
✔ Bracket/parenthesis matching  
✔ Syntax issues (print, scan, for, while, file functions, etc.)  
✔ Outputs formatted source data into a text file

---

## 🛠️ Features

This analyzer:

✅ Reads a C source file (`input.txt`)  
✅ Ignores empty lines and comments  
✅ Tracks keywords & built‑in functions  
✅ Checks matching brackets `{}`, `()`, `[]`  
✅ Counts variable types (`int`, `char`, `float`, etc.)  
✅ Checks basic syntax for functions like `printf`, `scanf`, etc.  
✅ Generates output in `output.txt` with line info
